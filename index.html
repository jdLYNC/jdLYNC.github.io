Diveboard

I created Diveboard as a better alternative to the job sites the industry currently uses. A MERN stack app, Diveboard integrates GoogleMaps, Places and Rest Countries APIs, bcrypt and JWT for user authentication, Axios and bluebird for http requests and bootstrap styling.

Approaching the build I committed significant time to planning, producing a comprehensive Trello board and wireframes. I practiced TDD on the back end and wrote 61 API tests. The React app is responsive with key pathways developed mobile first.

The primary challenge with this app was obtaining all the desired location information as GooglePlaces lacked regional data. This necessitated an additional request to Rest Countries during registration, for which managing the promises was a challenge.
